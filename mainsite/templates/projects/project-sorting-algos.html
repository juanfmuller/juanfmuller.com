PROJECT

Sorting Algorithm Visualizer -- responsive website

MY ROLE

Website Design

Code and Development

TOOLS USED

HTML5

CSS3

JavaScript

DATE

October 2020

As an electrical engineering student, my exposure to computer science concepts has been limited. I have taken it upon myself to learn these concepts through lecture and projects. This project focuses on understanding how sorting algorithms work through vizualisation.

The tools I decided to use were HTML and CSS for the responsive front-end and JavaScript for the logic and algorithms. I acomplished this by creating a specified ammount of divs and storing their height in an array. Afterwards, the array would be sorted using one of the specified algorithms. As it sorted the array, it also set the height of the div accordingly. Moving and coloring the divs at the specified speed.

I was able to develop a responsive website that illustrates different sorting algorithms. The number of bars on screen can be changed along with their thickness for better visuals. These two options can be randomized. Then there are two algorithms to choose from, quick sort and bubble sort. The speed the array is sorted can also be changed to better illustrate how the algorithm works.
