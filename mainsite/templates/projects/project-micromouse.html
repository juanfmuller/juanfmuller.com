Micromouse -- maze solving robot

DATE

January 2020

Micromouse is a competition sponsored by the Institute of Electrical and Electronics Engineers (IEEE), in which teams of 4 people compete to see who's robot will reach the center a maze first. We all started with the same printed circuit board and had to solder all the necessary components.For the microcontroller we used an Arduino, which allwed us to develop the software using C++ and external libraries.

For the robot to work correcty there were three main parts which had to be achieved. First the robot needed to move straight in between the walls, to do this we used a PID controller. The robot also needed to be able to avoid objects using the infrared sensors. Lastly, the robot had to remember where it had been and this was done with a floodfill algorithm.

Working alongside three peers, we were able to construct the robot and get it to run through the maze. I was in charge of developing the PID control which I developed using the wheel's encoders and an external library. Unfortunately, the competition was cut short due to COVID-19 and we were unable to put it to the race, however, the learning experience was great.
