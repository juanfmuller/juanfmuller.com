PROJECT

Xilinx -- digital circuit design

MY ROLE

Circuit designer

TOOLS USED

Xilinx

Verilog

DATE

April 2020

As en electrical engineer, my role is heavily focused on circuit design. I have decided to concentrate on computer engineering which focuses more on digital circuit design. A very useful tool is Xilinx, which allows for the design and testing of a digital circuit. This project is a digital design which I then tested using Verilog code.

Here, I designed a four bit register from bare flip flops. First, I created a standard register providing four outputs with four inputs. Using multiplexers I was able to design the multifunctional register, meaning it could hold, load, shift left, and shift right. To finally test the register, I made a verilog test fixture.

This project allowed me to get a better understanding of hardware is designed and tested. I also learned how to use Xilinx to prototype circuits, more specifically standard and multifunctional registers.
